## Retail Sales Performance Dashboard — Power BI
Made a sales analysis dashboard for a company's data in Power BI


Cleaned and sorted a superstore retail sales dataset first
Analyzed 4,000+ orders from the Superstore dataset to identify 
sales trends, top products, regional performance, and customer segments.

Tools used: Power BI Desktop, DAX
Key visuals: KPI Cards, Line Chart, Bar Chart, Map, Donut Chart, Top 10 Table
Dataset: Kaggle Superstore Dataset
